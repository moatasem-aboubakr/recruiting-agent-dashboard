# Recruiting Agent Dashboard

**Final Project – Data Integration and Visualization (Fall 2025)**

## Overview
An interactive data integration and visualization dashboard that loads live CSV data from Google Sheets and transforms it into actionable insights through KPIs, filters, and visual analytics.

## Key Features
- Live data integration from Google Sheets (CSV)
- Interactive KPIs (candidates, salary, experience, employment status)
- Advanced filtering (role, city, gender, education, salary, experience)
- Dynamic visualizations (charts, maps, correlations)

## Tech Stack
HTML5 · JavaScript · Tailwind CSS · AmCharts 5 · Google Sheets (CSV)

1. Clone the repo  
2. Open `index.html` in your browser  
3. Load the CSV via **Connect Sheet**

_No backend required._
